# Cybersecurity Education - Payload Learning Studio

## Overview
This is an **educational web-based simulator** designed to demonstrate how different types of malware (virus, ransomware, and remote access trojans) work conceptually. The application runs entirely in the browser with **no external connections, no actual malware, and no Kali Linux or C2 servers required**.

## Purpose
This tool is intended for:
- Cybersecurity students learning about malware behavior
- Security awareness training
- Understanding attack patterns in a safe, isolated environment
- Classroom demonstrations of payload concepts

## Features
- **Three interactive payload modules**: Virus, Ransomware, and RAT (Remote Access Trojan)
- **Live notepad-style logging** showing simulated attack chains
- **Educational code snippets** explaining each malware type
- **Clean, modern dashboard interface**
- **No actual malicious code** - purely educational simulation

## Modules

### 1. VIRUS.exe
- Demonstrates self-replication concepts
- Shows file infection simulation
- Explains polymorphic mutation techniques

### 2. RANSOMWARE.exe
- Simulates file encryption behavior
- Shows ransom note creation
- Demonstrates file scanning and targeting

### 3. RAT.exe (Remote Access Trojan)
- Illustrates backdoor persistence
- Shows keylogging concepts
- Demonstrates remote shell simulation

## How to Use
1. Open the HTML file in any modern web browser
2. Click any of the three payload buttons
3. Watch the live log panel for step-by-step simulation output
4. Use the "Clear Log" button to reset the notepad display
